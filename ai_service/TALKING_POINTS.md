# AI Talking Points

## What is Groq..

Groq is an AI inference platform used to generate fast AI responses using large language models.


## What does the AI service do..

The backend analyzes cloud security issues and generates security-focused explanations and recommendations.


## What cloud security issues were tested..

* Public S3 buckets
* Weak password policies
* Open security groups
* No MFA enabled
* Public API exposure



## What security protections were added..

* API rate limiting
* Input sanitization
* Prompt injection rejection
* Security headers
* Environment variable protection


## What testing was completed?

* Pytest unit testing
* OWASP ZAP scanning
* API endpoint testing
* Docker environment setup review


## What was learned during the project?

* Flask backend development
* API security basics
* AI integration workflow
* Docker basics
* Security testing process
