# AI Service Summary Card

## Project Name

Cloud Security Posture Manager — AI Service

## Main Endpoints

### POST /describe
Analyzes cloud security issues and explains risks.

### POST /recommend
Provides AI-generated security recommendations.

### GET /health
Checks backend service availability and status.


## Tech Stack
* Python
* Flask
* Groq API
* OWASP ZAP
* Docker
* Pytest
* ChromaDB
* Sentence Transformers
